# NightreignRelics

NightreignRelics is a powerful Minecraft plugin that introduces a tiered relic system with randomized passive effects, an addictive gacha acquisition mechanic, and deep technical integrations.

## ✨ Core Features

- **Relic System**: Obtain and equip relics of three different rarities: **Delicate**, **Polished**, and **Grand**.
- **Randomized Effects**: Over 20+ unique passive effects assigned randomly based on rarity.
- **Weapon Mastery & Buffs**: Effects in these categories are now non-stacking (only the strongest value applies).
- **Deep Combat Integration**: Cumulative damage calculations for melee, ranged, and elemental attacks.
- **Addictive Gacha**: A slot-machine style gacha system for obtaining relics with configurable costs.
- **Attribute Boosts**: Direct integration with Minecraft attributes (HP) and **AuraSkills** (Mana/FP).

## 🛠️ Requirements & Dependencies

- **Paper/Spigot 1.21.1+**
- **Vault** (Optional, for economy support)
- **AuraSkills** (Optional, for FP/Mana effects)
